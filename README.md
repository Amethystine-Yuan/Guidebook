# Welcome

欢迎！本手册为数字集成电路课程设计以及后续“毕业有芯”计划的指导手册，旨在为同学们从零开始学习后端提供有效的指导。本手册从 Linux 使用开始讲起，主要介绍了逻辑综合、物理综合、芯片级仿真等各个步骤，以及对应地在 Design Compiler 和 IC Compiler 中的各个脚本。希望同学们能够通过本手册对后端设计上有初步的理解。
